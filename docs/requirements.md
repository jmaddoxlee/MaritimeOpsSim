# MaritimeOpsSim Requirements

## Project Goal

Build an AI-assisted unmanned surface vessel mission simulator for safe search-and-rescue and maritime domain awareness scenarios.

## MVP Requirements

- The project shall have a C++ simulation engine.
- The project shall simulate at least one unmanned surface vessel.
- The project shall load at least one mission scenario from JSON.
- The project shall display a 2D maritime map using Raylib.
- The project shall generate telemetry logs.
- The project shall support basic mission success/failure logic.
- The project shall include a safe-scope disclaimer.

## Strong Portfolio Requirements

- A\* path planning
- Search-and-rescue mission logic
- Communication loss simulation
- SQLite mission logging
- Python analysis tools
- AI anomaly detection
- Mission risk scoring
- GoogleTest and Pytest tests
- GitHub Actions CI
- Demo video
- Downloadable release
